Показать, что расстояние Хемминга удовлетворяет аксиомам расстояния и может использоваться как метрика.

---
Должны выполняться:

1. $$d(x,y) \geq 0$$ - выполняется тривиально.
1. $$d(x, y) = d(y, x)$$ - если $$x = y$$, то во всех позициях символы совпадают $$\iff$$ и количество различающихся позиций = 0 $$\iff$$ $$d(x, y) = 0$$; если $$d(x, y) = 0$$, то нет ни одной позиции, где символы различаются, во всех позициях символы совпадают $$\iff x = y$$.
2. $$d(x, y) = 0 \iff x = y$$  - если расстояние между словами равно 0 $$\iff$$ в них нет отличающихся элементов.
3. $$d(x, y) \leq d(x, z) + d(z, y)$$:

Рассмотрим произвольную позицию i. Возможны случаи:

- Если $$x_i = z_i$$, то эта позиция не увеличивает $$d(x, z)$$.
- Если $$x_i \neq z_i$$, то возможны два варианта:
  * $$x_i \neq y_i$$ - тогда эта позиция учитывается в $$d(x, y)$$
  * $$y_i \neq z_i$$ - тогда эта позиция учитывается в $$d(y, z)$$
  (либо оба сразу)

В каждой позиции, где $$x_i \neq z_i$$, хотя бы одно из расстояний $$d(x, y)$$ или $$d(y, z)$$ содержит эту позицию.  
Следовательно, каждое различие между $$x$$ и $$z$$ покрывается хотя бы одним различием в парах $$(x, y)$$ или $$(y, z)$$.

Суммируя по всем позициям:

$$d(x, z) = |{i : xᵢ \neq zᵢ}| ≤ |{i : xᵢ \neq yᵢ}| + |{i : yᵢ \neq zᵢ}| = d(x, y) + d(y, z)$$

Доказать теорему

Код с минимальным расстоянием $d_{min}$ исправляет любые комбинации ошибок кратности $t \leq \lfloor (d-1)/2 \rfloor$, где $\lfloor x \rfloor$ - наибольшее целое, не превышающее $x$.


Передано $$c_i$$, принято $$x$$, $$d(c_i, x) = t$$.  
Декодирование вернет $$c_i \iff \forall c_j \neq c_i : d(c_j, x) > t$$.

**Неравенство треугольника:**

$$d(c_i, c_j) \leq d(c_i, x) + d(c_j, x) = t + d(c_j, x)$$

**По определению $$d_{\min}$$:**

$$d_{\min} \leq d(c_i, c_j) \leq t + d(c_j, x) \Rightarrow d(c_j, x) \geq d_{\min} - t$$

**Условие $$d(c_j, x) > t$$:**

$$d_{\min} - t > t \Rightarrow d_{\min} > 2t \Rightarrow d_{\min} \geq 2t + 1$$

**Выражаем $$t$$:**

$$t \leq \frac{d_{\min} - 1}{2} \Rightarrow t \leq \left\lfloor \frac{d_{\min} - 1}{2} \right\rfloor$$
