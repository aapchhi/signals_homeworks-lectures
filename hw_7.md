
Поле $\mathbb{F}_2$ состоит из двух элементов $\{0, 1\}$, арифметика в нём ведётся по модулю 2.  
В частности, $1 + 1 = 0$, откуда следует, что $1 = -1$ (вычитание эквивалентно сложению).

Поэтому:

$$
x^7 - 1 = x^7 + 1
$$

Подставим 
$x = 1$:

$$
1^7 + 1 = 1 + 1 = 0.
$$

Следовательно, $x = 1$ является корнем, а значит, двучлен $(x + 1)$ (в поле $\mathbb{F}_2$ это то же самое, что $(x - 1)$) является делителем.

Выполним деление столбиком или по схеме Горнера:
```
x^7 + 0x^6 + 0x^5 + 0x^4 + 0x^3 + 0x^2 + 0x + 1  |  x + 1
x^7 + x^6                                        |---------------
--------------------------------------------------  x^6 + x^5 + x^4 + x^3 + x^2 + x + 1
x^6 + 0x^5
x^6 + x^5
---------
x^5 + 0x^4
x^5 + x^4
---------
x^4 + 0x^3
x^4 + x^3
---------
x^3 + 0x^2
x^3 + x^2
---------
x^2 + 0x
x^2 + x
-------
x + 1
x + 1
-----
0

```
Таким образом:

$$
x^7 + 1 = (x + 1)(x^6 + x^5 + x^4 + x^3 + x^2 + x + 1).
$$

Рассмотрим:

$$
f(x) = x^6 + x^5 + x^4 + x^3 + x^2 + x + 1.
$$

Этот многочлен не имеет корней в 
$\mathbb{F}_2$


Над 
$\mathbb{F}_2$ 
все неприводимые многочлены малых степеней известны:

- Степень 1: $x$, $x+1$
- Степень 2: $x^2 + x + 1$
- Степень 3: $x^3 + x + 1$, $x^3 + x^2 + 1$

Поскольку $f(x)$ имеет степень 6 и не имеет корней, он должен раскладываться в произведение двух неприводимых многочленов степени 3 (либо быть неприводимым степени 6, но это легко проверить делением на известные кубические).

Проверим делимость на $x^3 + x + 1$:

```
x^6 + x^5 + x^4 + x^3 + x^2 + x + 1  |  x^3 + x + 1
x^6 + 0x^5 + x^4 + x^3               |----------------
-------------------------------------  x^3 + x^2 + 1
x^5 + 0x^4 + 0x^3 + x^2
x^5 + 0x^4 + x^3 + x^2
-----------------------
0x^4 + x^3 + 0x^2 + x
0x^4 + 0x^3 + 0x^2 + 0x
----------------------
x^3 + 0x^2 + x + 1
x^3 + 0x^2 + x + 1
-----------------
0

```
$$
x^6 + x^5 + x^4 + x^3 + x^2 + x + 1 = (x^3 + x + 1)(x^3 + x^2 + 1).
$$

Получаем разложение 
$x^7 - 1$ 
над 
$\mathbb{F}_2$:

$$
\boxed{x^7 - 1 = (x + 1)(x^3 + x + 1)(x^3 + x^2 + 1)}.
$$

Все три сомножителя **неприводимы** над $\mathbb{F}_2$:
- $m_0(x) = x + 1$ - линейный,
- $m_1(x) = x^3 + x + 1$ - кубический,
- $m_3(x) = x^3 + x^2 + 1$ - кубический 

Циклический код длины $n = 7$ над $\mathbb{F}_2$ полностью задаётся своим **порождающим многочленом** $g(x)$, который должен удовлетворять двум условиям:

1. $g(x)$ делит $x^7 - 1$;
2. Степень $g(x)$ определяет количество проверочных символов: $r = \deg g$.

Информационная длина кода вычисляется по формуле:
$$
k = n - \deg g.
$$

Все возможные делители $x^7 - 1$ получаются перемножением его неприводимых множителей в различных комбинациях. Исключая тривиальные случаи $g(x) = 1$ (код без избыточности) и $g(x) = x^7 - 1$ (нулевой код), получаем ровно 6 нетривиальных порождающих полиномов.

1. $g_1(x) = x + 1$
2. $g_2(x) = x^3 + x + 1$
3. $g_3(x) = x^3 + x^2 + 1$
4. $g_4(x) = (x + 1)(x^3 + x + 1) = x^4 + x^3 + x^2 + 1$
5. $g_5(x) = (x + 1)(x^3 + x^2 + 1) = x^4 + x^2 + x + 1$
6. $g_6(x) = (x^3 + x + 1)(x^3 + x^2 + 1) = x^6 + x^5 + x^4 + x^3 + x^2 + x + 1$

Для каждого кода вычислим $[n, k]$ и по возможности определим минимальное расстояние $d$.

- $\deg g_1 = 1 \Rightarrow k = 7 - 1 = 6$
- Код имеет параметры $[7, 6]$.
- Порождающая матрица состоит из всех векторов чётного веса (сумма битов по модулю 2 равна 0).
- **Минимальное расстояние**: наименьший вес ненулевого кодового слова — 2 (например, вектор с двумя единицами). Следовательно, $d = 2$.

**Тип кода**: **код с проверкой на чётность**

---

- $\deg g_2 = 3 \Rightarrow k = 7 - 3 = 4$
- Параметры: $[7, 4]$.
- Этот многочлен примитивен в $\mathbb{F}_2[x]$ (его корень порождает мультипликативную группу поля $\mathbb{F}_8$), следовательно, он определяет **код Хэмминга**.
- Для кода Хэмминга длины $2^r - 1$ с $r=3$ проверочными символами минимальное расстояние всегда равно 3.
- **Минимальное расстояние**: $d = 3$ (исправляет одну ошибку).

**Тип кода**: **код Хэмминга** $[7,4,3]$.

---

- $\deg g_3 = 3 \Rightarrow k = 4$
- Параметры: $[7, 4]$.
- Этот многочлен также примитивен (он двойственен к $x^3 + x + 1$). Он задаёт **эквивалентный код Хэмминга** — фактически тот же самый код с точностью до перестановки координат.
- $d = 3$.

**Тип кода**: **код Хэмминга** $[7,4,3]$.

---

- $\deg g_4 = 4 \Rightarrow k = 3$
- Параметры: $[7, 3]$.
- Этот код является **двойственным** к коду Хэмминга $[7,4,3]$, порождённому $x^3 + x^2 + 1$ (проверочный многочлен кода Хэмминга — произведение $(x+1)$ и сопряжённого кубического многочлена).
- Двойственный код Хэмминга называется **симплексным кодом** или **кодом максимальной длины**. Все ненулевые слова такого кода имеют одинаковый вес.
- Вес каждого ненулевого слова равен $2^{r-1} = 2^{2} = 4$ (при $r=3$ проверочных символах исходного кода Хэмминга).
- Следовательно, $d = 4$.

**Тип кода**: **симплексный код** $[7,3,4]$ (двойственный к коду Хэмминга).

---

- $\deg g_5 = 4 \Rightarrow k = 3$
- Параметры: $[7, 3]$.
- Аналогично предыдущему случаю, это двойственный код к другому представлению кода Хэмминга.
- Все свойства совпадают: $d = 4$.

**Тип кода**: **симплексный код** $[7,3,4]$.

---

- $\deg g_6 = 6 \Rightarrow k = 1$
- Параметры: $[7, 1]$.
- Информационное слово состоит всего из одного бита (0 или 1). При кодировании оно повторяется 7 раз:
  - $0 \rightarrow (0,0,0,0,0,0,0)$
  - $1 \rightarrow (1,1,1,1,1,1,1)$
- Следовательно, минимальное расстояние $d = 7$ (вес ненулевого слова).

**Тип кода**: **код с повторением** $[7,1,7]$.

| Порождающий полином $g(x)$                         | Степень $\deg g$ | $[n, k]$  | $d$ | Тип кода                         |
|---------------------------------------------------|------------------|-----------|-----|----------------------------------|
| $x + 1$                                           | 1                | $[7, 6]$  | 2   | Проверка чётности                |
| $x^3 + x + 1$                                     | 3                | $[7, 4]$  | 3   | Код Хэмминга                     |
| $x^3 + x^2 + 1$                                   | 3                | $[7, 4]$  | 3   | Код Хэмминга (эквивалентный)     |
| $(x+1)(x^3+x+1) = x^4+x^3+x^2+1$                 | 4                | $[7, 3]$  | 4   | Симплексный код (двойственный)   |
| $(x+1)(x^3+x^2+1) = x^4+x^2+x+1$                 | 4                | $[7, 3]$  | 4   | Симплексный код (двойственный)   |
| $(x^3+x+1)(x^3+x^2+1) = x^6+x^5+\dots+1$         | 6                | $[7, 1]$  | 7   | Код с повторением                |

Разложение $x^7 - 1$ над полем $\mathbb{F}_2$ даёт три неприводимых множителя, комбинируя которые, можно построить 6 различных циклических кодов длины 7.
